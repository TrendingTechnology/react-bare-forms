React library to build forms with Bootstrap 4 styling.
#### Install
```
npm install react-bare-forms
```

#### Usage



#### Bootstrap 4
React Bare Forms only includes bootstrap 4 components required to style forms.
The following components are used:
- Forms
- Buttons
- Alerts
